## ADDED Requirements

### Requirement: 点赞功能
系统 SHALL 提供点赞按钮和本地点赞状态管理。

#### Scenario: 点赞操作
- **WHEN** 用户点击点赞按钮
- **THEN** 系统切换点赞状态（点赞/取消点赞），更新 UI 显示，状态仅保存在本地

#### Scenario: 点赞数显示
- **WHEN** 视频展示时
- **THEN** 系统显示当前点赞数，点赞后数字 +1，取消点赞后数字 -1

### Requirement: 评论 UI
系统 SHALL 提供简单的评论输入和展示界面。

#### Scenario: 评论输入
- **WHEN** 用户点击评论按钮
- **THEN** 系统弹出评论输入框，用户可输入评论内容

#### Scenario: 评论展示
- **WHEN** 用户提交评论
- **THEN** 系统将评论添加到本地评论列表并展示，评论仅保存在本地

### Requirement: 视频信息展示
系统 SHALL 在视频画面上叠加展示视频标题、作者等信息。

#### Scenario: 信息叠加显示
- **WHEN** 视频正在播放
- **THEN** 系统在视频画面底部叠加显示视频标题、作者名称，不影响视频播放

#### Scenario: 操作按钮布局
- **WHEN** 视频正在播放
- **THEN** 系统在视频画面右侧显示点赞、评论、分享等操作按钮
